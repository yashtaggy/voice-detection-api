# AI-Generated Voice Detection API

A FastAPI-based REST API that detects whether a voice sample is **AI-generated** or **Human** for the following languages:

- Tamil
- English
- Hindi
- Malayalam
- Telugu

The API accepts **Base64-encoded MP3 audio**, validates input strictly, and returns a JSON response.
